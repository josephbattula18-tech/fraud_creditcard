# fraud_creditcard
# Bank Churn Prediction using Machine Learning

## 📌 Overview

This project predicts whether a customer will churn (leave the bank) using machine learning models.

## 🚀 Features

* Data preprocessing and cleaning
* Handling categorical variables
* Random Forest model training
* Evaluation using ROC-AUC, F1-score, confusion matrix

## ⚠️ Key Learning

* Identified and removed **data leakage**
* Improved model performance realistically

## 📊 Results

* Accuracy: 95%
* ROC-AUC: 0.986
* Good performance on churn detection

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python model.py

## 📁 Dataset

Kaggle Bank Churn Dataset (mention link)
